# Monfette Conjecture v3 — p–e Law & Safe Primes

This repository contains an interactive program (Tkinter + Matplotlib) that:
- counts safe primes up to x (up to 10¹⁰ depending on hardware),
- compares π_SG(x) with the Hardy–Littlewood prediction,
- visualizes the error and GRH-type envelopes,
- automatically generates a Markdown report with figures and analysis.

-----------------------------------------------------------------------

This project is a numerical research tool.  
It does not aim to prove conjectures, but to provide:

- reproducible numerical data,
- clear visualizations,
- an exact combinatorial foundation (the p–e law),
- an exploration tool for cryptographers and number theorists.
